# Frontend test by Joanna Kalema 

In terms of the instructions on how to use:

- Unzip the file 
- Place the file in a the correct file that your code editor can access.
- Using code editor of choice open the file (all the relevant files should be there)
- In the terminal run npm install to install the relevant dependencies onto device
- Once complete run npm start which open the web page onto any browser open 










This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
